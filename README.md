# SCP5K-Mod-Tools
Modding tools and utilities for SCP: 5K

Currently, Includes an editor plugin that ensures the correct plugin data is set when exporting mods.
This is NOT a mod SDK, you will need to unpack the game yourself. You can find information about the unofficial community SDK [Here](https://www.scp5k-modding.wiki/docs/sdk/sdksetup/).
